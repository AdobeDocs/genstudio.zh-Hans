# 面向性能营销人员的Adobe GenStudioBeta发行说明

这些说明重点指出了在截至9月19日的一周内为性能营销人员修复和增强功能提供的重大Adobe GenStudio。

## 新增功能和增强功能

* **与Adobe Experience Manager Assets集成**。 现在提供了对Adobe Experience Manager Assets的只读访问权限。<!-- GS-2432 -->

* **对更新模板工作流的增强**。  现在，更新模板的用户可选择要为其使用模板的渠道。<!-- GS-4029 -->

* **已改进创建页面加载性能**。 未使用的依赖项已从页面加载进程中移除。<!-- GS-3630 -->

* **多节电子邮件支持**。 编辑器现在可以生成包含多个部分和图像的电子邮件。 他们还可以重新生成所生成电子邮件的特定片段（例如，标题）。<!-- GS-2436 -->

* **在创建期间，在桌面视图和移动设备视图之间进行切换**。 用户现在可以在桌面视图和移动设备视图之间切换以预览电子邮件体验变体。<!-- GS-4314 -->

* 内容现在会生成具有相对于原始资源维度的裁切维度的图像。<!-- GS-3150 -->

* 用户现在可以选择生成的图像变体，并在创建工作流期间使用调整裁切功能裁切它们。<!-- GS-5538 2342 -->

* 已批准体验的详细信息视图现在会显示缩略图图像以及该体验中引用的所有资产的状态。<!-- GS-3783 -->

## 已知问题

GenStudio for Performance Marketers GA版本中计划解决以下已知问题。

* 间歇性延迟问题会影响某些[!DNL Create]画布操作。<!-- GS-5203 -->

* 电子邮件生成导致电子邮件不完整。 **解决方法**：刷新页面并重新生成。<!-- GS-5209 -->

* 可以上传模板，但无法看到模板。 **解决方法**：上载填充了&#x200B;**[!UICONTROL 促销活动]**&#x200B;字段的资产。 然后，再次上传模板。<!-- GS-4815 -->

* 用户还必须登录两次Facebook才能登录渠道元广告帐户。 **解决方法**：在登录到渠道元广告帐户之前，注销Facebook。<!-- GS-4806 -->

### 其他增强功能和修复的问题

* 现在，拖放操作在提示区域中可按预期运行。<!-- GS-3977 -->

* 更正了使用Tab键在左侧导航栏上的元素中导航时出现的问题。 以前，需要多次单击才能从一个元素导航到下一个活动元素。 <!-- GS-2639 -->

* GenStudio现在可在用户编辑体验名称时保存该名称，同时加载体验。<!-- GS-5242 -->

* 用户现在可以成功编辑体验标题。 以前，在用户尝试编辑标题文本后，标题文本默认为原始文本。<!-- GS-5246 -->
* 现在，在多部分电子邮件创建期间，选定的图像会按预期在画布上渲染。<!-- GS-5263 -->

* [!DNL Content]体验详细信息页面中的所有字符串现已本地化。<!-- GS-5016 -->

* 用户现在可以删除在[!DNL Products]中打开其详细信息视图的产品。<!-- GS-5057 -->

* GenStudio在搜索未产生匹配结果时显示的消息已得到改进。<!-- GS-4544 -->

* 搜索筛选器值的`aria-label`属性值现在已按预期翻译。<!-- GS-5388 -->

* 用户现在可以成功删除[!DNL Create]画布提示区域中的重复资产。 <!-- GS-5233 -->

* 帐户筛选器现在可按预期用于体验、资源和属性。<!-- GS-4812 -->

* 元广告模板的字体问题已得到解决，从而提高可读性和可访问性。<!-- GS-5354 -->

* 对草稿标题的更改现在会按预期保留。 以前，标题在编辑后恢复到默认名称。<!-- GS-2951 -->

#### 模板修复

* 现在，调整大小功能可按预期用于元广告模板中的多个图像。 以前，GenStudio不会为所有选定的模板调整图像大小。<!-- GS-4696 -->

* 删除模板现在会按预期刷新[!DNL Content]页面。<!-- GS-5397 -->

* 用户现在只能从下拉菜单选择[!DNL Personas]、[!DNL Brands]或[!DNL Products]的值。 以前，_模板上传_&#x200B;对话框允许用户输入任何[!DNL Persona]、[!DNL Brand]或[!DNL Product]名称。<!-- GS-5072 5071-->

* 现在，在模板上传过程中，**[!UICONTROL 返回]**&#x200B;按钮处于禁用状态。<!-- GS-5358 -->

* [!DNL Create]选择模板详细信息视图中的所有字符串现在都已本地化。<!-- GS-5025 -->

## 早期Beta版本

早期Beta版本包括以下亮点和修复。

### 高亮

* [!DNL Create]内容选择器已重构，以改进资源加载。<!-- GS-2586 -->

* GenStudio现在支持在[!DNL Insights]表和图库视图中预览媒体资源的选项。 视频缩略图包含一个带有静音选项的&#x200B;**播放**&#x200B;按钮。<!-- GS-4398 -->

* instagram和Facebook渠道准则已合并到元品牌准则中。

* [!DNL Create]画布导航元素已简化。 [!DNL Create]登陆页面显示左侧导航面板，但用户现在使用&#x200B;**[!UICONTROL 后退]**&#x200B;按钮从其他[!DNL Create]工作区导航到此共享空间。

* 导航元素已得到增强，可在执行整个产品（包括以下产品领域）中的任务时支持用户聚焦：

   * [!DNL Content]中的资产、体验、模板详细信息
   * [!DNL Insights]中的体验、资产、属性详细信息
   * [!DNL Brands]中的品牌详细信息
   * 产品和角色中的产品和角色详细信息

* 用户无需再单击&#x200B;**[!UICONTROL 刷新]**&#x200B;按钮即可查看[!DNL Content]中体验的更新。

* _Experience Details_&#x200B;页面现在将外部资源缩略图呈现为HTML。

* 添加或删除Assets和体验后的UI延迟已得到改进。

* 模板预览现在包含更具描述性的默认文本。 请参阅[自定义模板](https://experienceleague.adobe.com/en/docs/genstudio/user-guide/content/templates/customize-template#template-preview)。

* **基于百分比的验证分数**：品牌验证现在以百分比的形式显示品牌验证分数，而不是通过/未通过值。 （固定8/16）

* **已更新品牌提取界面**：品牌提取现在以百分比显示提取流程的完成情况。 （固定8/16）

* **在提取期间增量加载品牌**：品牌指南现在增量加载到用户界面中。 （固定8/16）

* **多节电子邮件创建**：用户现在可以创建由单独的标题、图像、正文和CTA元素组成的电子邮件。 （固定8/16）

* **元广告调整大小**：编辑器可以调整元广告长宽比。 （固定8/16）

* **有限的[!DNL Insights]登录帐户**： [!DNL Insights]登录现在仅支持每个客户一个帐户。 （固定8/16）

### 其他增强功能和修复的问题

* [!DNL Create]画布现在正确呈现元广告中的图像。 （固定9/13） <!-- GS-4864 -->

* 尽管元广告画布预览和导出的视图之间可能存在差异，但导出的体验会按预期工作。 （固定9/13） <!-- GS-4492 4401 -->

* 现在，上传的图像包含预期的智能标记。 （固定9/13） <!-- GS-4856 -->

* 元广告导出CSV文件现在按预期包含图像。 以前，ZIP文件包含CSV导出文件和NULL文件，而不是图像。 （固定9/13） <!-- GS-5107 -->

* 用户现在可以按预期在“模板详细信息”视图&#x200B;**[!UICONTROL 上载者]**&#x200B;字段中输入文本。 以前，加载图标会阻止用户输入文本。 （固定9/13） <!-- GS-4887 -->

* 删除品牌后，用户不再被重定向到品牌的“详细信息”视图。 （固定9/13） <!-- GS-2663 -->

* 在发送变体以供审阅和批准时，编辑者不再收到以下错误：`You have no access to view comments on this Object`。 （固定9/13） <!-- GS-5140 -->

* 更新了审阅和批准工作流使用的电子邮件模板。 （固定9/13） <!-- GS-5239 -->

* 现在，在加载模板选择器期间发生网络错误时，GenStudio会显示一条错误消息。 （固定9/13） <!-- GS-4682 -->

* 解决了从资产、体验或模板卡片导航到选定对象时出现的问题。 （固定9/13） <!-- GS-4390 -->

* 现在，从“创建画布”中打开&#x200B;_添加Assets_&#x200B;弹出窗口时，该窗口已本地化。 （固定9/13） <!-- GS-4867 -->

* 现在为重新生成的变体触发品牌验证。 以前，如果编辑者重新生成了现有草稿的变体，则不会触发验证。 （固定9/13） <!-- GS-3971 -->

* _添加Assets_&#x200B;弹出窗口现在已按预期本地化。 （固定9/5） <!-- GS-3834 -->

* 元广告体验模板的缩放问题已得到解决。 （固定9/5） <!-- GS-4174 -->

* 多部分电子邮件CSV导出文件中的文本字段现在按预期排序。 （固定9/5） <!-- GS-4013 -->

* 当用户反复按&#x200B;**退格键**&#x200B;以擦除搜索字段文本时，[!DNL Content]搜索字段不再消失。 （固定9/5） <!-- GS-4543 -->

* 当协作者在评论中添加`@`提及时，GenStudio性能营销人员现在会按预期加载用户。 以前，未加载用户并显示错误： `Unable to load users. Refresh the page`。 （固定8/29） <!-- GS-4113 -->

* 当编辑者在提示区域中创建电子邮件期间单击&#x200B;**选择内容**&#x200B;时，GenStudio不再显示&#x200B;**发生错误**&#x200B;消息。<!-- GS-4879 -->

* _体验详细信息_&#x200B;页面馈送放置名称现在指定了Facebook或Instagram馈送。 （固定8/16）

* 现在，当用户从&#x200B;_资源概述_&#x200B;视图导航到&#x200B;_资源详细信息_&#x200B;视图时，大图像和视频的裁剪是一致的。 （固定8/16）

* 在用户登录之前，“属性”屏幕搜索结果计数不再显示`0 of`。 （固定8/16） <!-- GS-3665 -->

* 单击&#x200B;**[!UICONTROL [!DNL Insights]]** > **[!UICONTROL 资产]**&#x200B;计数字段不再清除搜索和筛选设置。 （固定8/16） <!-- GS-3476 -->

* 当用户尝试在[!DNL Insights]视图中输入凭据时，GenStudio显示错误。 （固定8/29） <!-- GS-4689 -->

* 由于ACP存储平台问题，品牌指南上传失败。 （固定8/22） <!-- GS-4369 -->

* 在电子邮件创建过程中，提示区域[!DNL Brands]下拉菜单会在[!DNL Brands]列表的末尾显示一个旋转图标。 （固定8/22） <!-- GS-4077 -->
